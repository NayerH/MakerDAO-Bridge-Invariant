# MakerDAO Bridge Invariant

## Description

This bot detects if the total supply of DAI minted on L2s, Optimism and Arbitrum, surpases the DAI locked in the respective escrow contracts on Ethereum

## Supported Chains

- Ethereum
- Optimism
- Arbitrum
